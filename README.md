# 👻 Unity Horror Game

Unityで制作した一人称視点のホラーゲームです。暗闇の中を探索し、謎を解きながら脱出を目指します。

---

## 🎮 ゲーム概要

- **タイトル**：おばちゃんの家（後で変更可能）
- **ジャンル**：ホラー / 探索 / 
- **対応プラットフォーム**：Windows /
- **プレイ時間**10分程度

---

## 🧰 使用技術

- Unity 2022.3 LTS
- C#
- Post Processing Stack
